UnmockWe fake it while you make it

Unmock helps you test the business logic of your API integrations by creating unreasonably effective simulations of external APIs and microservices.
Get started

When should I use Unmock?

Unmock is useful whenever you are testing code that calls an external API. This can be a microservice, a backend server, a third-party API like Stripe or Contentful, or an analytics service like Sentry or Segment.
